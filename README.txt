INSTALL ON WINDOWS
Just extract the archive and run qTwister.exe.

SHORT MANUAL
This application is meant to use by Twister players.
There are two modes of play: manual and automatic.
In manual mode you should click the big red "Twist!" button to
generate next move.
Automatic mode allows you to generate moves by timer with
interval given. To turn this mode on select "Play" item 
of Play menu. To stop automatic play, select Play->Stop or click
the big red "Stop" button. Every move is displayed on screen by
picture of body part (left/right foot/hand) and color. Also, every move is accompanied by recorded voice.

SETTINGS
In settings dialog (Play->Settings) one can:
 - set up automatic moves generation interval;
 - turn sound on and off;
 - allow/disallow the generation of same moves in a row;
 - adjust the arrow animation speed;
 - choose interface and voice language;
 - choose between male and female voice.

SOUNDS AND TRANSLATIONS
In this version there are two interface and voice languages - English and Russian. Russian voices were recorded manually, English ones - with the Espeak text-to-speach tool. In sounds/en/espeak.sh you may found command lines I used to generate English voices.
If you want to add a localization or voice - don't hesitate to write me.

УСТАНОВКА ПОД WINDOWS

КРАТКОЕ ОПИСАНИЕ
Программа предназначена для любителей играть в Твистер.
Играть можно в двух режимах: ручном и автоматическом.
В ручном режиме для генерации следующего хода нужно просто
нажать на красную кнопку с надписью "Твист!"
Автоматический режим позволяет герерировать ходы по таймеру,
то есть через заданные промежутки времени. Для его включения 
необходимо выбрать пункт "Игра" одноименного меню, а для выключения - либо выбрать пункт "Остановить", либо нажать красную
кнопку "Стоп". Результат каждого хода отображается на экране в виде конечности (левой или правой руки или ноги) и цвета, а также озвучивается.

НАСТРОЙКИ
В диалоге настроек (Игра->Настройки) можно:
 - задавать длительность интервалов между автоматическими ходами (в секундах);
 - включать и выключать звук;
 - разрешать/запрещать генерацию нескольких одинаковых ходов подряд;
 - регулировать скорость анимации стрелки;
 - выбирать язык интерфейса и звуков;
 - выбирать между мужским и женским голосом, озвучивающим ходы.

ЗВУКИ И ПЕРЕВОДЫ
В настоящее время доступно два языка интерфейса и озвучки ходов - русский и английский. Русская озвучка записана самостоятельно, английская - сгенерирована с помощью программы espeak. В директории sounds/en/ находится файл espeak.sh, с помощью которого и была сгенерирована озвучка.
Если у кого-то есть желание добавить еще один язык или еще одну озвучку - обращайтесь.
